    InitializeOnEnable

    Controls whether underlying FMOD object should be automatically initialized on load. Set this to false when using WebGL to avoid interacting with FMOD before it has properly loaded. Set it to true on all other platforms. When set to false initialization can be triggered in 2 ways:

    • When something calls Initialize() explicitly, e.g. WebGLInitializer (this is the easiest and preferred way).
    • On getting or setting Volume
