    Playing FMOD one-shot events

    • FMODEvent as a ScriptableObject, which references a single FMOD event, and allow simple one-shot playback. These instances are very convenient to reference from e.g. UnityEvents, and can make adding simple sound effects and UI sounds very easy without having to add StudioEventEmitter components everywhere.
