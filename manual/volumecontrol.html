    Persistent Volume Control

    A common task for any game is to allow the player can adjust audio volume from a settings menu and persist this setting between play sessions.

    This package contain several components for making this easy.

    • VCAVolumePreference and BusVolumePreference are ScriptableObjects for managing FMOD VCAs and buses respectively. Each instance manages a single VCA or bus, and persists the current volume to PlayerPrefs.
    • VolumeSlider is a VisualElement, which renders a Slider bound to a VCAVolumePreference or BusVolumePreference for building settings menus.

    See the sample for how to build a simple audio settings menu.
